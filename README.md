# Ithaka Backend  
 The backend for Ithaka's CRM tool  

### Installation  
1. Clone the repo in an appropriate place using

        git clone
2. Install the dependencies by running

        npm i  

### Running Locally  
There are two ways of running the backend locally, either by using node or the handy nodemon tool for quick bugfixing  

1.      nodemon index.js  
2.      node index.js  

### Documentation  
To refer to the documentation, first run the server and go to

    https://localhost:3000/docs
